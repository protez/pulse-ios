Pulse for iPhone and iPad

Building an unofficial client for Neudesic's Pulse product for use on the iPhone and iPad.

View the project on GitHub

Introduction

Welcome to the website for the unofficial iPhone and iPad client for Neudesic Pulse. The goal of this website is to document the design and construction of the Pulse client application and to provide end-user documentation and helpful tips and tricks for interacting with Neudesic Pulse and the Pulse client.

Please note that this is an unofficial project. It's a hobby project that has three primary goals:

  1. Learn how to program against the Pulse API
  2. Experiment with ideas for developing iOS applications around Pulse
  3. Serve as an example to other Pusle users to demonstrate how the Pulse API can be used to integrate with Pulse

As a Neudesic employee, Pulse is a product that I use very frequently during my daily activities. I use it to follow news about the company as well as to collaborate with customers and co-workers on projects. Pulse can help to facilitate communication and discussion in distributed teams much more effectively than email.
